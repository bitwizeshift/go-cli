# Examples

The subdirectories in this folder contain independent examples of how the
`go-cli` library can be used to create command-line applications. Each example
is self-contained and can be run independently.

* [Simple](./simple) - A simple `example-cli` showcasing all of the different
  functionality of the library in a succinct package.

* [Custom Flags](./custom-flags) - A more complex example that demonstrates how
  to use the library to create a command-line application with custom
  flag-definitions to control behaviors in a testable, sharable way.

* [Prompts](./prompt) - A small example application that demonstrates the
  different "prompt" types that can be used to gather user input in a
  command-line application.

* [Progress](./progress) - An example application that showcases all of the
  different progress-bars and spinners that can be used to provide feedback to
  the user during long-running operations in a command-line application.
